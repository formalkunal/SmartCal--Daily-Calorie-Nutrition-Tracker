## Project Overview
SmartCal is a Java-based desktop application developed as part of a GUVI project.
It helps users track daily calorie intake and basic nutritional values such as
protein, carbohydrates, and fats.

## Technologies Used
- Java
- Java Swing (GUI)
- JDBC
- MySQL
- Apache NetBeans / IntelliJ IDEA

## Features
- User login system
- Food calorie and nutrition tracking
- Input validation
- Database connectivity
- User-friendly graphical interface
